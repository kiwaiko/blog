### Map

```JavaScript
const names = [ "a", "b", "c", "d",]
// names.map(anonymous function to map)
names.map((name) =>(){console.log(name)})
// mutate and add elements to names
// if the function returns a value it cam mutate and change values


names.map((name) =>(){
    return name + "!"
    })
// names = [ "a!", "b!", "c!", "d!",]

// or return jsx element
names.map((name) =>(){
    return <h1>{name}<h1/>
    })
```

### Filter

```javascript
// remove "a" from names
const names = [ "a", "b", "c", "d","a","a"]
names.filter( (name) => () {
    return (name!="pedro")
    })
```
